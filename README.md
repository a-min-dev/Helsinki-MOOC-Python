*These selections demonstrate specific logical milestones:*

* **[gift_tax_calculator.py](./Portfolio/gift_tax_calculator.py)** - Tiered tax bracket logic and formatted rounding

* **[find_second_occurrence.py](./Portfolio/find_second_occurrence.py)** - Finds the second occurrence of a
user-provided substring in a user-provided word and outputs
the index of the start of the second substring
* **[chessboard.py](./Portfolio/chessboard.py)** - The chessboard function
prints out alternating rows of 0s and 1s
* **[spruce_tree.py](./Portfolio/spruce_tree.py)** - The program uses two functions, line and spruce, to output a centered
spruce tree diagram
* **[longest_words_in_list.py](./Portfolio/longest_words_in_list.py)** - The program uses a function, all_the_longest,
to return a new list of the all the longest strings in the original list
